海洋生物 Levy 飞行模拟程序 - 快速启动指南
一、文档说明
本文档用于指导拆分后的 Python 模拟程序快速启动与使用，包含目录结构、依赖安装、运行步骤、功能说明及常见问题排查，适用于中小型项目部署场景。
二、目录结构
确保所有文件放在 同一目录 下，目录结构如下：
plaintext
marine_levy_simulation/  # 项目文件夹
├── main.py              # 主执行程序（含全局定义）
├── ui_scripts.py        # UI界面脚本（交互逻辑）
├── calc_scripts.py      # 运算脚本（核心模拟逻辑）
└── requirements.txt     # 依赖包清单
三、依赖安装
1. 环境要求
Python 版本：3.8 及以上（推荐 3.9-3.11）
操作系统：Windows/macOS/Linux（Windows 需确保 Python 已添加环境变量）
2. 安装步骤
打开终端 / 命令行，切换到项目目录：
bash
cd /path/to/marine_levy_simulation  # 替换为实际项目文件夹路径
执行依赖安装命令：
bash
pip install -r requirements.txt
（若安装失败，可尝试更换镜像源：pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple）
四、运行步骤
启动程序：在项目目录下执行命令
bash
python main.py
程序启动后，将弹出【文件选择窗口】，按以下流程操作：
选择地图文件：点击【Browse】选择 Excel 格式的地图文件（后缀.xlsx）
选择温度文件（仅加权模式需选）：加权模式下点击【Browse】选择匹配的温度 Excel 文件
选择方向模式：
加权模式：50% 随机 + 50% 温度偏好，需配套温度文件
随机模式：完全随机方向，无需温度文件（自动禁用温度文件选择框）
配置参数：
Number of Agents：生物个体数量（默认 500，可调整）
Max Steps：最大模拟步数（默认 100000，可调整）
Random Seed：随机种子（可选，填写后模拟结果可复现）
寿命限制：勾选后可设置生物最大寿命（单位：年）
洋流模式：选择是否启用洋流影响（默认启用）
禁用方向：点击方向按钮（↑↓←→）可禁用对应移动方向（按下为禁用状态）
点击【Run Simulation】启动模拟，终端将显示进度条
模拟完成后，自动弹出【结果交互窗口】，可：
查看模拟路径图（左侧）：支持缩放、平移（使用 matplotlib 工具栏）
导出 PNG：点击【Export PNG Image】选择保存路径，可自定义存活个体颜色
查看统计信息：右侧显示步数、存活数、死亡类型分布等
关闭窗口：点击【Close Window】结束程序
五、核心功能说明
模拟功能：
生物移动：基于 Levy 飞行模型，支持随机模式和温度加权模式
环境交互：支持地形（G/B/Y 类型）、洋流、温度偏好等环境因素
死亡机制：包含补给超时死亡、寿命到期死亡、超出边界死亡
交互功能：
方向禁用：可单独或组合禁用北 / 南 / 东 / 西移动方向
模式切换：支持洋流启用 / 禁用、寿命限制启用 / 禁用
结果导出：支持透明背景 PNG 导出，自定义存活个体颜色
六、常见问题排查
1. 启动失败：ModuleNotFoundError
问题原因：依赖包未安装或安装不完整
解决方法：重新执行依赖安装命令，确保所有包安装成功；若仍失败，手动安装缺失包（如pip install 缺失的包名）
2. Excel 文件读取失败
问题原因：
文件格式错误（非.xlsx 后缀）
地图文件与温度文件尺寸不匹配（加权模式下）
文件路径包含中文或特殊字符
解决方法：
确保文件后缀为.xlsx（不支持.xls 格式）
加权模式下，温度文件行数 / 列数需与地图文件完全一致
将文件移至无中文、无空格的路径下（如D:/sim_data/map.xlsx）
3. 模拟过程卡顿
问题原因：生物数量过多（如超过 1000）或最大步数过大（如超过 200000）
解决方法：减少生物数量（建议 500 以内）或降低最大步数（建议 100000 以内）
4. PNG 导出失败
问题原因：保存路径包含中文或特殊字符，或无写入权限
解决方法：选择无中文、无空格的保存路径（如D:/sim_results/paths.png），确保文件夹有写入权限
5. 窗口闪退
问题原因：Python 版本过低（低于 3.8）或 matplotlib 后端冲突
解决方法：升级 Python 至 3.8 及以上；若仍闪退，在main.py开头添加以下代码：
python
运行
import matplotlib
matplotlib.use('TkAgg')
七、注意事项
地图文件格式：Excel 文件需无表头，直接以网格形式存储地形数据（支持 G/B/Y 类型及洋流方向数据）
温度文件格式：Excel 文件需无表头，尺寸与地图文件一致，存储温度数值
性能优化：若电脑配置较低，建议减少生物数量（200-300）和最大步数（50000）
结果复现：如需复现相同模拟结果，需填写固定的随机种子（如 12345）
